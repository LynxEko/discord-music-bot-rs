# DJ Maderium

Discord music bot made in Rust


## compiling and running

Before anything you need a `config.toml` file it should have the following format:

```toml
[discord]
bot_token = ""
guild_id = ""

[youtube]
api_key = ""
project_id = ""

type = ""
private_key_id = ""
private_key = ""
client_email = ""
client_id = ""
auth_uri = ""
token_uri = ""
auth_provider_x509_cert_url = ""
client_x509_cert_url = ""
universe_domain = ""
```
